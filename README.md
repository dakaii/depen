## Getting Started

### Prerequisites

- Rust
  ```sh
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dakaii/depen.git
   ```
2. Run the tests
   ```sh
   cargo test
   ```
